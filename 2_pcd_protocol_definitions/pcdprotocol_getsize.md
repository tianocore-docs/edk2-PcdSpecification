## PCD_PROTOCOL.GetSize()

#### Summary

Retrieves the size of the value for a given PCD token.

#### Prototype

```c
typedef
UINTN
(EFIAPI *PCD_PROTOCOL_GET_SIZE)(
  IN UINTN  TokenNumber
  );
```

#### Parameters

**_`TokenNumber`_**

The PCD token number.

#### Description

Retrieves the current size of a particular PCD token. If the **_`TokenNumber`_** is
invalid, the results are unpredictable.
